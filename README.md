## Waybar Integration

To add ScrollMPRIS as a custom module in [Waybar](https://github.com/Alexays/Waybar), insert the following snippet into your Waybar config:

```json
"custom/ScrollMPRIS": {
    "return-type": "json",
    "exec": "~/.config/waybar/scripts/ScrollMPRIS",
    "escape": true,
    "on-click": "playerctl play-pause"
},
```
